Python Project: Mastermind

Voor dit project van de code taal Python moesten we het spel 'Mastermind' namaken

Dit is een klassiek denkspel waarin de speler een code moet raden die door het programma is gegenereerd.
Het doel is niet alleen om het correcte getal te raden, maar ook om de juiste volgorde te vinden.

Spelregels

1. Het programma genereert een willekeurige code van vier cijfers (bijvoorbeeld 1234).
2. De speler moet de code raden door vier cijfers in te voeren.
3. Na elke gok geeft het programma feedback:
- Een '+' geeft aan dat een cijfer correct is en op de juiste positie staat.
- Een '-' geeft aan dat een cijfer correct is, maar op de verkeerde positie staat.
- Geen teken betekent dat het cijfer volledig incorrect is.
4. Het spel eindigt wanneer de speler de juiste code en de juiste volgorde heeft geraden.
5. Na afloop moet het programma het aantal pogingen weergeven en een gepersonaliseerd bericht afhankelijk van het aantal pogingen.

Programma-eisen

1. Gebruik functies om de code modulair te maken.
2. Implementeer foutafhandeling om ongeldige invoer van de speler te voorkomen.
3. Geef de speler de mogelijkheid om het spel opnieuw te spelen na het raden van de code.
4. Voeg een moeilijkheidsniveau toe waarbij de lengte van de code kan variÃ«ren (bijv., easy: 3 cijfers, medium: 4 cijfers, hard: 5 cijfers).

Uitdagingen

1. Implementeer een timer om de speeltijd te meten en houd een record bij van de snelste tijden.
2. Voeg kleur toe aan de consoleoutput om de feedback visueel aantrekkelijker te maken.
